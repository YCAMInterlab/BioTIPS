## バイオ・リサーチ・オープンデイ「キッチンからはじめるバイオ」

### 概要
YCAMバイオ・リサーチでは、近年飛躍的に発達するバイオテクノロジーの応用可能性を、芸術表現や教育、地域といった多様な切り口で模索しています。2016年度は、全6回のテーマを設定してリサーチを行い、その過程を展示やワークショップ、バイオラボツアーを通じて紹介する場として「オープンデイ」を開催しました。
